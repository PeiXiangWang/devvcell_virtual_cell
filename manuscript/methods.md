# Methods

## Teacher Construction

The internal developmental map is represented by native moscot temporal transport plans. The teacher provides barycentric developmental velocity, transition entropy and pseudo-lineage structure. M0b_ot_interpolation is treated as an oracle-like OT reference rather than a competitor that the finite-agent model must beat.

## Primary Agent Model

M5_ot_swarm is retained as the primary mechanistic model. It combines OT teacher velocity with local swarm alignment, cohesion and separation terms in latent expression space. More complex modules were audited but not retained for the main claim because birth/death, memory and CCI did not receive sufficient support.

## v1.2 Topological-Rule Audit

The v1.2 analysis tests whether branch nucleation can be explained by a small fixed number of developmental neighbours, inspired by topological interaction principles in collective behaviour but adapted to single-cell latent dynamics.

Neighbour rules were evaluated as follows:

- Topological-k: k = 1, 2, 3, 5, 7, 9, 12, 15, 20, 30 nearest neighbours in latent space.
- Metric-radius: latent-distance quantile radii q = 0.01, 0.03, 0.05, 0.10, 0.20.
- Random-k: random neighbours matched to the topological k values.
- Label-k: neighbours restricted to the same lineage label when available.
- Mixed-k: kNN on a combined latent and fate-probability representation.

For each rule, branch event windows were identified from changes in lineage separation, velocity alignment and fate entropy. The recorded order parameters were local velocity alignment, branch cohesion, lineage separation, fate entropy, branch imbalance, local density and cell count.

## Susceptibility and Minimal Model

Developmental velocity fluctuations were correlated across graph-distance bins. The analysis reports correlation length and susceptibility around branch windows but does not assert finite-size scaling. A minimal pairwise topological model estimates local label-coupling and velocity-alignment parameters and predicts the sign of branch-order-parameter shifts without training on branch event labels.

## Perturbation Diagnostic

Local perturbation propagation was simulated as an in silico graph-response diagnostic from high-entropy pre-branch cells. The readout is affected fraction by graph distance, response attenuation and branch-composition shift. This is not an experimental perturbation result.

## Evidence Rules

Results are tiered by teacher backend, negative controls, external time-series support and clone-aware evidence. L2 clone-aware support remains failed, so branch nucleation is retained as a computational hypothesis rather than clone-resolved evidence.
