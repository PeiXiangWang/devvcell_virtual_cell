# SwarmLineage-OT Developmental Branch-Window Order Parameters

SwarmLineage-OT uses native OT-inferred pseudo-lineage maps as supervision for finite-agent virtual-cell dynamics. The retained story is not that the agent model surpasses OT, but that it exposes a measurable branch-window order-parameter signature in developmental time-series data.

Across the internal native teacher, E1 MouseGastrulationData, and the developmental atlas analysis, the primary signature is transient condensation-before-divergence: lineage separation contracts around a detected branch window and then re-expands. New datasets are explicitly tiered by teacher backend, negative-control cleanliness and baseline specificity.

The current atlas-level tier is `weak`: internal/E1 remains the main support; new time-series support is weak or unresolved. Clone-aware fate-diversification support remains not established, and unsupported modules are excluded from the main claim.
