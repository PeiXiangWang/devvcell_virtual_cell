# DevGuard 数据注册表

## 主数据原则

主数据应使用独立公开小鼠 embryo/gastruloid 数据。第一篇 RDEG 的 `scLine_pro.h5ad` 只能作为 supplementary sanity check，不进入主图 1-5 的核心证据链。

配置文件：

```text
config/devguard/datasets_mouse.json
```

## 统一 obs schema

所有 processed H5AD 必须包含：

| 字段 | 含义 |
|---|---|
| `cell_id` | cell ID |
| `dataset_id` | dataset ID |
| `species` | species, mainline must be `Mus musculus` |
| `system` | embryo / gastruloid / organoid |
| `time_point` | E day / gastruloid day / hour |
| `time_numeric` | numeric developmental time |
| `condition` | control / perturbation |
| `perturbation_name` | KO gene / treatment |
| `perturbation_type` | genetic / chemical / pathway / environmental |
| `dose` | dose or NA |
| `duration` | treatment duration or NA |
| `sample_id` | embryo / organoid / donor ID |
| `batch` | batch |
| `cell_type` | cell type annotation |
| `lineage` | broad lineage |
| `is_control` | bool |
| `is_perturbed` | bool |

Processed outputs:

```text
data/processed/devguard/{dataset_id}.h5ad
results/devguard/dataset_metadata/{dataset_id}_metadata.csv
```
